# crypto-trading-sql-project
A MySQL database project for a simplified cryptocurrency trading system, featuring user assets, orders, trade history, wallet valuation, automatic balance updates, and profit analysis
##features
- user asset queries
- user order queries
- trade history queries
- wallet value calculation
- automatic wallet balance updates using triggers
- profit analysis using stored procedures
- coin metadata storage using json
